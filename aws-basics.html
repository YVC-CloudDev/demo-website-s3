AWS Basics for beginners

Goal: understand AWS building blocks and set up a safe student account.

Learning objectives

  • Create an AWS account and secure it
  • Understand region and availability zone basics
  • Know when to use S3, EC2, and Lambda
  • Enable budget alerts to stay inside free tier

Prerequisites

  • Email address and phone number
  • Bank card for account verification
  • A notebook to track what you create
AWS may charge if you exceed limits. Always remove unused resources.

Core concepts

Regions

Regions are physical locations (for example, eu-west-1). Pick one region and stay consistent in tutorials.

Availability Zones

Multiple isolated zones inside each region. They help build reliable apps.

Service model

AWS provides many services. You start with a few and combine them into one project.

Beginner service map

S3

Store static files such as HTML, CSS, images, and documents.

EC2

Create virtual servers when you need full control of an operating system.

Lambda

Run code on demand without managing a server.

Budget alert quick setup

Use this checklist before starting any lab:

  1. Open AWS Billing dashboard
  2. Go to Budgets and create a monthly cost budget
  3. Set budget value to $5
  4. Add email notifications at 60% and 90%
  5. Confirm alert emails are arriving
If you complete this once, you can safely explore more AWS tutorials.
